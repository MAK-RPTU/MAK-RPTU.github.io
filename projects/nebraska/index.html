Nebraska · Tiago & Tiago Pro — LLM-Grounded Manipulation

Object classification and 6D pose estimation for cluttered-scene manipulation on Tiago and Tiago Pro, with GPT-3.5 Turbo and LLaMA integrated as foundation-model interfaces for natural-language grasping behaviour.

Tiago 6D pose YOLOv8 / v11 PoseCNN GPT-3.5 LLaMA ROS 2

Highlights

  • Implemented object classification and 6D pose estimation (YOLOv8 / v11, PoseCNN) for manipulation in cluttered scenes on Tiago and Tiago Pro; built an evaluation pipeline comparing simulated and real-camera inputs.
  • Integrated GPT-3.5 Turbo and LLaMA on a Tiago for natural-language manipulation — wrapped third-party model APIs in a structured ROS 2 interface, enabling swappable foundation-model comparison for grasping.
  • Set up Isaac Sim environments for Tiago and Tiago Pro and built ROS 2 service / client interfaces for object attach / detach across simulated and real pick-and-place.
  • Taught the fundamentals of NVIDIA Isaac Sim to a professor through structured presentations and hands-on simulation sessions.
